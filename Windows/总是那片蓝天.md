# 简介
始终同一片蓝天...是一个迷人的没有像他一样的他视觉小说。两个年轻的灵魂，以自己的方式努力地感觉世界的一个扣人心弦的奇幻爱情故事。惊人的艺术作品和悦耳的配乐，是不可错过的青年才俊！

设想一下，由于神秘的情况下，你被远送到地中海岛国学习。虽然你已经习惯了远行，但即使你没有经历过的这样的一个小岛......像基拉这样的一个女孩。始终同一片蓝天脱颖而出作为一个真正独特的，海边的冒险;一个永恒的爱情，神秘和人性的故事。
